# Changelog

All notable changes to Flowforge will be documented in this file.

## 0.1.1 - 2026-07-24

- Fixed creating the first shape in an empty Mermaid file.
- Added inserting a dropped shape directly onto an edge.
- Fixed undo restoring the previous canvas state.
- Improved adaptive arrow direction and Shift multi-selection.

## 0.1.0 - 2026-07-24

- Added richer canvas editing: durable edge routing, connection controls, node styling, and expanded shape support.
- Improved external-change recovery, source synchronization, inspector controls, and editor accessibility.
- Refined the modern editor layout, release assets, and VS Code packaging metadata.
